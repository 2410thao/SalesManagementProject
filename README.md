# SalesManagementProject

Vĩnh Long", cấp dưới của "TP. Vĩnh Long" là các Phường. Tiểu thương (hoặc các điểm bán tạp hóa) là người trực tiếp sử dụng phần mềm để quản lý hoạt động kinh doanh của mình.
